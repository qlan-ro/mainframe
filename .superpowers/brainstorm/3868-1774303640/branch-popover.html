Interaction Details

  • Search: Filters branches as you type. If no match, shows "Create branch [name]" option
  • Click branch: Checks out that branch (with dirty-tree warning if uncommitted changes)
  • Fetch: Runs git fetch, refreshes branch list
  • Pull: Pulls current branch from remote
  • Merge/Rebase: Opens a sub-view to pick the source branch, then runs the operation
  • Conflict state: If merge/rebase hits conflicts, popover shows conflict file list + abort button
