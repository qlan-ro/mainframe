05

Typography

Two system stacks do all the work — no web fonts. SF Pro Text (via -apple-system) for UI and prose; SF Mono for code, metadata, counts, and identifiers. Tight tracking at display sizes, neutral at body.

Type scale 8 rungs · FS

Consolidated to 8 named rungs in FS — granular where dense UI lives (10 / 11 / 12 / 13), opening up for display (15 / 17 / 22 / 28). Every inline fontSize snaps to a rung; the old fractional half-points (11.5, 12.5, 14.5 …) are gone. Floor is 10px (mono micro-metadata); body sits at 13px.

Weights

Regular (400) is reserved for muted/inline contexts; 500 is the resting UI weight, 600 marks active/selected state and buttons, 700 titles and headings, 800 the brand mark only.

Radius, elevation & spacing

Soft, consistent rounding and two shadow tiers — a tight popover lift and a deep modal float — both pairing a blur with a hairline ring so edges read on any surface.
